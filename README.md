# AI × Web3 School Learning Workspace

Personal learning journal and proof-of-work for AI × Web3 School.

## Quick Links
- Handbook: https://aiweb3.school/zh/handbook/
- WCB Program: https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning: https://web3career.build/zh/programs/AI-Web3-School#tab=learning
- WCB Agent API: https://web3career.build/llms.txt

## Privacy & Security
This workspace may be published to a **public** GitHub repository.
Do **NOT** commit:
- API keys / tokens / secrets
- wallet mnemonic / private keys
- personal sensitive data
- private meeting links

Use local environment variables (e.g., `WCB_AGENT_SECRET_API_KEY`) for secrets.

## Structure
- `profile.md`: learner profile and goals
- `learning-plan.md`: weekly plan (minimum / recommended / challenge)
- `daily/`: daily study + check-in drafts
- `tasks/`: task notes and deliverables
- `experiments/`: mini projects and technical experiments
- `handbook-feedback/`: issue/suggestion feedback to handbook
- `hackathon/`: hackathon ideas and build logs
- `submissions/`: links and submission records
- `templates/`: reusable note templates

## Daily Workflow
1. Check WCB Learning page
2. Read one handbook section
3. Write `daily/YYYY-MM-DD.md`
4. Draft check-in content
5. Submit manually on platform
6. Save links/evidence back to daily note
