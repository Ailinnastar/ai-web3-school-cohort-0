# AI × Web3 Personalized Learning Plan (v1)

## Positioning
你当前最优路线：**AI 强项驱动 + Web3 基础补齐 + 可展示作品沉淀**。
目标不是“学完所有概念”，而是持续产出 proof-of-work，逐步识别你的职业优势位。

## 6-Week Plan

### Week 1–2: Foundation Bridge
- AI baseline refresh (Agent / RAG / tool use)
- Web3 basics: wallet, network, smart contract mental model
- Output:
  - 4 daily notes
  - 1 glossary note (`tasks/glossary-ai-web3.md`)

### Week 3–4: Practical Integration
- Chain-aware context, Web3 tool use, agent workflow
- Mini experiment 1:
  - "Wallet activity explainer" or
  - "AI × Web3 concept explainer bot"
- Output:
  - 1 experiment brief in `experiments/`
  - 1 short demo README

### Week 5–6: Career Signal Building
- Pick one direction:
  - A) AI x Web3 product research
  - B) AI-assisted builder workflow
  - C) Hackathon prototype track
- Output:
  - 1 portfolio-quality project summary
  - 1 bilingual self-intro draft (`tasks/self-intro.md`)

## Weekly Rhythm
- Mon/Tue/Thu: core study + mini output (45–90 min)
- Wed/Fri: light review + check-in draft (30–45 min)
- Weekend: deep work (2–3h)

## Daily Paths
### Minimum (45 min)
1. Handbook 1 section (20 min)
2. Note + summary (15 min)
3. 3-sentence EN practice (10 min)

### Recommended (60–90 min)
1. Handbook + WCB task check (25 min)
2. Practical micro-task (25–40 min)
3. Daily check-in draft (10–20 min)

### Challenge (120+ min)
1. End-to-end mini build block
2. README/documentation update
3. Feedback note to handbook
